搭建CTP接口量化交易Web服务

本文最后更新于：2022年11月26日 下午

CTP是上期技术推出的一套可供程序调用的交易接口，在期货和ETF期权量化交易中被广泛使用。由于期货市场的开放性，个人用户进行穿透认证后即可使用，门槛较低。本文将其中的交易接口做进一步的轻量化封装，仅供研究参考。

一、CTP接口Python Web服务

基于ctpwrapper和ctp_client的CTP轻量级Python Web服务封装，可查询期货、期权行情以及进行下单交易。

前提：研究的话注册一个simnow账号，实盘需要联系期货公司申请CTP穿透认证(推荐使用VNPY，一般在自己电脑上登录一下就OK)，这样便可愉快的玩耍[亏钱]了

  • HTTP请求Tick行情
  • HTTP请求期货期权合约代码
  • 内外盘期货实时行情
  • 新闻资讯、经济数据
  • HTTP下单(限价、市价单的开仓与平仓)
  • HTTP获取持仓以及资金
  • 每日定时自动启动(白天8:40，夜盘20:40)

二、安装

  • 首先安装cython等: pip install -U cython aiohttp apscheduler
  • 然后安装ctpwrapper: pip install -U ctpwrapper
  • 最后安装web服务器: pip install sanic<22
